# Quizmaster_Appdev1
Basic quiz master application for the AppDev1 course.

This application allows users to create and participate in quizzes. Quizzes can be created by admin, and can be taken by any user. The application keeps track of the scores for each user.

The application is built using Flask and SQLAlchemy, with a PostgreSQL database. The frontend is built using Bootstrap and Jinja2 templates.


## Features

* User authentication and authorization
* Quiz creation and management
* Quiz taking and scoring
* User profile management


## Technical Details

* Built with Flask and SQLAlchemy
* Uses a PostgreSQL database
* Frontend built with Bootstrap and Jinja2 templates
